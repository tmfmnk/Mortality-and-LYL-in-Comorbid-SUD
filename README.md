# Mortality and Life-Years Lost Following Subsequent Physical Comorbidity in People with Pre-existing Substance Use Disorders: A Registry-based Retrospective Nation-wide Cohort Study of Hospitalised Individuals

This repository contains the complete analytical code and raw outputs of the study on all-cause mortality and life-years lost following subsequent physical comorbidity in people with pre-existing substance use disorders, based on data from the Czech nation-wide registers of all-cause hospitalizations and all-cause deaths. The content of this repository is as follows:

<b>1. Sample data.</b>  

Due to the sensitive character of the data, it cannot be published or shared with external subjects without a permission granted by the Czech Institute of Health Information and Statistics. However, we generated sample data with the structure of the data used in this study.

<b>2. Raw outputs.</b> 

Contains all the unedited tables and graphs, as generated in R. 

<b>3. Analytical dode.</b> 

The analytical code is divided into separate files, covering the data preparation, matching, fitting of survival models, generating the number of outcomes, and computing the life-years lost.
